Correction Guide

Quality Checklist

Download Markdown

If you like checklists, here is one you can use:

Decisions based on evidence

  • I compared the draft against the original image.
  • Every major edit is supported by the original image.
  • I did not invent missing content.
  • I did not use the binarized draft as a reference.

Binary correctness

  • The final image contains only foreground and background.
  • There are no gray pixels.
  • There are no semi-transparent pixels.
  • There are no colored annotation marks.

Content preservation

  • Text strokes are not accidentally removed.
  • Punctuation is preserved.
  • Diacritics and accents are preserved.
  • Thin lines and small symbols are preserved when visible in the original.
  • Broken strokes were repaired only when supported by the original.

Noise removal

  • Isolated speckles were removed when they are not content.
  • Page-border artifacts were checked.
  • Shadows and paper texture were not kept as foreground.
  • Stains were removed.
  • Bleed-through was removed.

File integrity

  • Filename follows the required pattern.
  • Image dimensions match the input.
  • Final file is saved in the required format.
  • Notes file exists for uncertainty or special decisions.
