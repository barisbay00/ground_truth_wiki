Correction Guide

Common Error Types

False foreground

This is when the background becomes black.

Common sources:

  • paper texture
  • dust
  • shadows
  • stains
  • compression artifacts
  • page edges

Fix: paint those pixels white if the original does not support them as content.

False background

This is when foreground becomes white.

Common sources:

  • faint ink
  • thin strokes
  • weak punctuation
  • over-aggressive noise filtering
  • broken characters

Fix: paint those pixels black if the original supports them as content.

Stroke thickening

Foreground becomes too thick compared with the original. This should be extremely rare, otherwise, something must have went terribly wrong.

Fix: there is no fix, stop working on this image. Just note it down for review and inform me.

Stroke thinning

Foreground becomes too thick compared with the original. This should also be rare, otherwise, yet again, something must have went terribly wrong.

Fix: there is no fix, stop working on this image. Just note it down for review and inform me.

Lost small marks

Dots, punctuation, accents, and diacritics may be erased by filtering.

Fix: inspect small marks carefully and then add appropriately if necessary.

Border artifacts

Page edges create black strips or ragged shapes.

Fix: remove them with a large size pencil tool or the select tool.
