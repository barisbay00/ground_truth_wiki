File Handling and Naming

Recommended folders

Use this structure when possible:

project/
  01_original/
  02_binarized_draft/
  03_working/
  04_ground_truth/
  05_review_notes/

Naming pattern

Use a stable pattern so files can be matched automatically:

<image_id>_original.<ext>
<image_id>_draft.png
<image_id>_ground_truth.png
<image_id>_notes.md

Example:

page_00042_original.jpg
page_00042_draft.png
page_00042_ground_truth.png
page_00042_notes.md

No need for personal initials

Dimensions

The ground truth should normally have the same pixel dimensions as the draft and original. Do not crop, rotate, resize, deskew, or shift content.
