Task Workflow

Annotation Workflow

Step 1: Prepare files

Make a working copy of the binarized draft using the structure in File Handling and Naming.

Step 2: Open images side by side

Place the original image and working binary image next to each other.

Practical setup

For example, place the original on the left and the working binary image on the right. Move through the image systematically:

  1. top to bottom
  2. left to right
  3. borders and margins
  4. dense text areas
  5. faint or damaged regions
  6. final full-page scan

Step 3: Correct false foreground

Remove black pixels that represent noise, stains, shadows, dust, page edges, or other background artifacts.

[!warning] Before deleting tiny marks, compare with the original image. A dot may be punctuation or a diacritic.

Step 4: If necessary, restore false background

Add black pixels where the draft erased real foreground that is visible in the original image.

Examples:

  • missing punctuation
  • missing diacritics
  • disconnected character parts

[!warning] Do not guess missing content and please restore only what is clearly visible in the original.

Step 5: Check local consistency

When zoomed in, always ask:

  • Does the correction match the original?
  • Did I accidentally thicken or reshape the mark?
  • Did I introduce pixels?
  • Did I remove meaningful content?

Step 6: Check global consistency

Zoom out and inspect the whole image. Look for inconsistent cleanup, accidental brush marks, and missed border artifacts (there are usually many border artifacts you can clean them with a wide brush or the select tool).

Step 7: Save final output

Save the final image in the PNG format because it is lossless and widely supported.

Run Quality Checklist and then save the final image in the PNG format (because it is lossless).
