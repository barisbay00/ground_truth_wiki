Handoff Procedure

We have decided to progress with the work packages on a rolling basis so it would be great if you could upload the ground truths to the cloud as soon as you are done with them.

Suggested final package

04_ground_truth/
  page_00042_ground_truth.png

05_review_notes/
  page_00042_notes.md/.txt/.doc

If using software with project files, keep the working file separately:

03_working/
  page_00042_working.xcf

For GIMP, .xcf is useful as a working format, but the submitted ground truth should still be the agreed image format, .png.
