Home

This is a mini wiki that explains how to work a binarized, noise-filtered image into a reliable ground truth image for testing binarization algorithms.

What?

Okay, so, some of the wiki here is not essential knowledge and you do not need to know about most of these concepts to fulfill this task. The task is as simple as making sure the black and white images reflect the original manuscript images as closely as possible. It can be done in paint. All you really need to know is...

The goal

The final ground truth should preserve visible foreground content and remove false foreground noise, without inserting content that is not in the original image. See Foreground and Background.

What will I do?

You will receive two images for each item:

  • an original image in color
  • a binarized draft in black and white that has already been noise filtered

Your task is to improve the binarized draft by comparing it against the original image until the draft reflects the original image as much as possible. The ideal final output should represent the best binary reference answer for that image.

See Navigation Map for templates and checklists and especially important sections (02 Workflow and 03 Corrections)

What do I need?

Any image manipulation tool of your choosing (e.g., GIMP, paint, preview). Almost all operating systems today are shipped with at least a basic image manipulation software, so, chances are, you already have one installed.

Example workflow

  1. Open the original image and binarized draft side by side.
  2. Remove false foreground noise.
  3. Restore missing foreground that is visible in the original (should not be the case normally).
  4. Save using the agreed filename and format.
  5. Run the Quality Checklist.

Help line
