Troubleshooting

Common Problems

I am not sure whether a mark is text or noise

Compare against the original image and check nearby similar marks. If it remains uncertain, record it in notes.

See Ambiguity Handling.

My brush creates gray edges

Switch to a pencil tool, disable antialiasing or use a hard brush.

I accidentally edited the original

No worries, we do have back-ups. you can just stop using that file, restore it from the original source and continue from a clean copy.

The output dimensions changed

Return to the working copy before resizing if possible. Try exporting again without resizing. If dimensions are already changed and cannot be restored, you can ask me for help.

I removed punctuation by mistake

Use undo if possible. If there is no way of undoing it, then just let me know and I can fix it later.

The file looks correct but may not be binary

Use a tool that can inspect pixel values or convert the final to strict black/white. GIMP can help with this.
