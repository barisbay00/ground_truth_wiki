Software Guides

Tool Settings Checklist

Download Markdown

Before editing, configure your tool so corrections stay binary.

Brush or pencil

  • hard edge
  • no softness
  • no transparency
  • no antialiasing if configurable
  • black color is pure black
  • white color is pure white

Canvas

  • same dimensions as input
  • no accidental crop
  • no resize
  • no rotation
  • result is in one layer (ignore if you don't use layers)

Export

  • PNG format and no compression
